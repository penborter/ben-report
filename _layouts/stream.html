--- layout: default --- {% comment %} Renders the stream feed: post/note/photo/review entries from _data/stream.yml, merged in date order with file-based thoughts from _thoughts/. The merge uses the sort-string trick — one row per item: "||", sorted descending. Thoughts live as Markdown files in _thoughts/. They have no stream.yml entry — they're picked up from `site.thoughts` directly and their date is the YYYY-MM-DD prefix on the filename. Title (optional) and `link` (optional manual title link, e.g. "/books#…") come from front matter. `link` is used rather than `url` because Jekyll computes `doc.url`. IMPORTANT: `pages:` must be declared LAST in `_config.yml` `collections:` so feed.md (using this layout) renders after all the collections it references — otherwise doc.content reads as raw source. {% endcomment %}
